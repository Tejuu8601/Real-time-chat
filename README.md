# Real-Time Chat Application

A responsive **Real-Time Chat App** built with **React.js** (frontend) and **Node.js + WebSockets** (backend).  
This project was developed as part of the **CodTech Internship**.

---

## Features

- Real-time messaging using **WebSockets**
- Responsive chat interface
- Smooth message animations
- Timestamps and avatars for messages
- Light / professional theme
- Separate sections with background colors
- Enter name before joining chat

## Setup Instructions

### 1. Backend
```bash
cd backend
npm install
node server.js

Server runs on ws://localhost:5000

2. Frontend
cd frontend
npm install
npm start

React app runs on http://localhost:3000

Usage

Enter your name to join the chat.

Send and receive messages in real-time.

View message timestamps and avatars.
